This project presents an AI-inspired NDVI-based vegetation health and density assessment of Wayanad District, Kerala, using Sentinel-2 satellite imagery.

## Study Area
Wayanad District, Kerala, India (Western Ghats region)

## Data Used
- Sentinel-2 Surface Reflectance imagery
- Administrative boundary data (DIVA-GIS / GADM)

## Methodology
- Satellite data pre-processing in Google Earth Engine
- NDVI computation using Band 4 (Red) and Band 8 (NIR)
- AI-inspired rule-based vegetation classification
- Area statistics generation
- Final cartographic layout prepared in QGIS

## Tools & Software
- Google Earth Engine
- QGIS

## Outputs
- AI-based NDVI vegetation health and density map
- Area-wise vegetation statistics
- GIS-ready final map layout
